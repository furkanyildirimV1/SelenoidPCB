# SolenoidPCB

TR: Bu proje 12V bir selenoid valfin Arduino veya görev bilgisayarından alınan sinyalle anahtarlanması için PCB tasarımıdır.

ENG: This project is a PCB design for switching a 12V solenoid valve with Arduino or a mission computer.

## Features
- Low-side MOSFET switching
- Reverse polarity protection (P-channel MOSFET ideal diode)
- TVS diode + fuse + capacitors protection

**Status:** Prototyping stage
