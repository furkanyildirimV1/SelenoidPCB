# SelenoidPCB
Selenoid valf sürücü devresi PCB tasarımı || Selenoid valve circuit PCB design
Bu proje 12V bir selenoid valfin Arduino veya görev bilgisayarından alınan sinyalle anahtarlanması için devrenin PCB tasarımıdır || This project is a PCB for switching 12V selenoid valve with Arduino or Mission computer.
Selenoid valf MOSFET low side ile anahtarlama devresi  ||  Selenoid valve MSOFET low side switching circuit
Sigorta, TVS diyot, kapasitörler ve P kanal MOSFET ile PCB koruma || PCB protection with TVS diode, fuse, capacitors, P channel MOSFET
