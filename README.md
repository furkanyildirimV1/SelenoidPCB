# SelenoidPCBTR
Selenoid valf sürücü devresi PCB tasarımı  
Bu proje 12V bir selenoid valfin Arduino veya görev bilgisayarından alınan sinyalle anahtarlanması için devrenin PCB tasarımıdır.
Selenoid valf MOSFET low side ile anahtarlama devresi. 
Sigorta, TVS diyot, kapasitörler ve P kanal MOSFET ile PCB koruma.
**DURUM**: Prototip aşaması

# SelenoidPCBENG
Selenoid valve circuit PCB design
 - This project is a PCB for switching 12V selenoid valve with Arduino or Mission computer
 - Selenoid valve MSOFET low side switching circuit
 - PCB protection with TVS diode, fuse, capacitors, P channel MOSFET
**STATUS**: Prototyping stage
